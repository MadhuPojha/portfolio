# Madhu Priyadarshni Ojha - Portfolio

## Project Description

A modern, responsive portfolio website built with React and Vite, showcasing professional experience, skills, projects, and education. This portfolio demonstrates full-stack development expertise with a focus on clean code, performance optimization, and user experience. The site is deployed on GitHub Pages and features smooth scrolling, dynamic components, and an intuitive navigation system.

## Features

- **Responsive Design** - Mobile-first approach with seamless experience across all devices
- **Dark Theme** - Modern dark color scheme with gradient accents for visual appeal
- **Smooth Navigation** - Fixed navbar with smooth scroll anchors to different sections
- **Scroll-to-Top Button** - Quick navigation button that appears when scrolling down
- **Project Showcase** - Interactive project cards with descriptions, technologies used, and GitHub links
- **Skills Display** - Organized skill categories with icons from React Icons
- **Experience Timeline** - Detailed work experience with responsibilities and achievements
- **Education & Certifications** - Comprehensive list of certifications and courses
- **Contact Links** - Direct links to LinkedIn, GitHub, and email
- **Optimized Performance** - Built with Vite for fast development and optimized production builds
- **CSS Modules** - Scoped styling to prevent style conflicts

## Installation Instructions

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Git

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/MadhuPojha/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```
   npm install
   npm install react-icons
   ```

3. **Run the development server**
   ```
   npm run dev
   ```
   The application will start at `http://localhost:5173`

4. **Build for production**
   ```
   npm run build
   ```
   This creates an optimized build in the `dist` folder

5. **Preview the production build**
   ```
   npm run preview
   ```

6. **Deploy to GitHub Pages**
   ```
   npm run deploy
   ```

### Technologies Used
- **Frontend**: React 19, Vite
- **Styling**: CSS Modules, CSS Variables
- **Icons**: React Icons
- **Build Tool**: Vite
- **Deployment**: GitHub Pages
